Le web du geek

Vincent De Oliveira · @iamvdo

C'est quoi le web?

  • Site web / Site applicatif / Mobile
  • Aucune différences de technologie
    • Site d'infos
    • Webmail
    • Intranet
    • Facebook
  • Trois piliers
    • Nom de domaine (DNS)
    • URI (HTTP)
    • Langages: HTML, CSS, JavaScript, etc

Spécificités du web

  • Environnement non contrôlé
    • Clients web multiples
    • Tailles d'écrans, résolutions (Desktop, mobile)
    • Connexion (Filaire, WIFI, 4G, Edge)
  • Accessibilité des données

Langages Client

  • HTML
    • Contenu des pages
    • Textes, images, vidéos, etc.
  • CSS
    • Mise en forme graphique
  • JavaScript (JS)
    • Interactivité
    • Logique applicative

Langages Serveur

  • PHP
  • JAVA
  • JavaScript (Node.js)

Architecture classique

  • Client
    • Fais des requêtes au serveur via l'interface
    • Affiche le résultat
  • Serveur
    • Traite les requêtes du client
    • Fournit / stocke les données

Architecture moderne

  • AJAX (chargement asynchrone)
  • Client
    • Fais des requêtes au serveur via le code
    • Traite les données
    • Affiche / modifie le résultat
  • Serveur
    • Traite les requêtes du client
    • Fournit / stocke les données

Application web

  • API JS (HTML5)
  • Client
    • Traite les données
    • Stocke les données
    • Affiche / modifie le résultat
    • Communique avec d'autres clients (P2P)
  • Serveur
    • Charge moins importante

Application installable

  • Firefox OS, Windows 8
  • Le site web devient l'OS

Application packagée

  • Rendre l'appli native
  • PhoneGap

Environnement client

  • Graphique
    • Canvas (WebGL), SVG, CSS, RWD, etc.
  • Contenu
    • Elements d'UI, Offline, AppCache, History, etc.
  • Données
    • IndexedDB, WebStorage, WebSockets, WebRTC, etc.
  • Système
    • Geolocalisation, FileAPI, Battery, SMS, etc.
